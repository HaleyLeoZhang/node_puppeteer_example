## 起步

> 安装依赖

~~~bash
# 安装依赖包
npm install
# 安装 gulp 
npm install -g gulp
# 生成兼容es5语法的node文件
gulp start
~~~

> 配置

请到目录 `es6/conf/db/mysql.js` 配置 `mysql`  
本次表结构请看目录 `sql/`


### 目录介绍

`es6` 源代码路径
`es5` node 可直接运行路径

### 运行

~~~bash
# 运行应用 
node es5/app.js
~~~
